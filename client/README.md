# RSML Annotator - React Frontend

Vite + React frontend for the RSML Annotation tool.

## Installation

```bash
cd client
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:5173` (Vite default) and proxy API requests to the backend.

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Features

- ✅ Batch and File selection
- ✅ RSML Annotation with live preview
- ✅ Audio playback for each segment
- ✅ Save RSML data to backend
- ✅ Keyboard shortcuts modal
- ✅ Same UI/UX as original HTML version
